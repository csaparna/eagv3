# New Tab Quest ⚡

A high-difficulty, Gemini-powered 2D platformer Chrome extension that overrides your New Tab page. You must complete a procedurally generated level to access your browser shortcuts and search!

## 🎮 Game Features

- **Gemini-Powered Themes**: Every level features a unique theme (name, colors, background) generated by Google Gemini 1.5 Flash.
- **Procedural Level Design**: Algorithmic path-building ensures every level is difficult but guaranteed to be playable within the game's physics limits.
- **Physics-Based Movement**: Precise jumping and movement controls designed for a challenging experience.
- **Dynamic Hazards**: On-platform lava and gap-fill lava zones that force you to plan your route.
- **Integrated Dashboard**: A sleek search and shortcut dashboard revealed only after victory or giving up.

## 🛠️ Technology Stack

- **Core**: Vanilla JavaScript, HTML5 Canvas, CSS3.
- **AI**: Google Gemini API (Model: `gemini-1.5-flash-lite-preview`).
- **Styling**: Modern dark-mode aesthetics with glassmorphism and smooth animations.

## 🚀 Setup Instructions

### 1. Configure the API Key
You need a Google Gemini API key to use the AI features.
1. Create a file named `secrets.js` in the root directory.
2. Add your API key:
   ```javascript
   const API_KEY = "YOUR_GOOGLE_GEMINI_API_KEY_HERE";
   ```

### 2. Load the Extension
1. Open Chrome and navigate to `chrome://extensions/`.
2. Enable **Developer mode** (toggle in the top right).
3. Click **Load unpacked**.
4. Select the directory containing this project.

### 3. Start Playing
Open a new tab in Chrome, and the challenge will begin!

## ⌨️ Controls

- **Left / Right Arrows**: Move
- **Up Arrow**: Jump
- **Give Up Button**: Exit the game and reveal the dashboard immediately.

## 🧠 Behind the Scenes: The Stage Pipeline

1. **Stage 1 (Generation)**: Gemini generates a theme (colors and name).
2. **Stage 2 (Layout)**: A JavaScript algorithm builds a connected chain of platforms ensuring jumps are physically possible.
3. **Stage 3 (Validation)**: Algorithmic lava placement ensures hazards don't block necessary paths.
